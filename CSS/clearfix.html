等高框

有相同高度的浮动框：

Box 1

Some content, some content, some content

Box 2

Some content, some content, some content

Some content, some content, some content

Some content, some content, some content

这个例子不是很灵活。如果可以保证框中始终有相同数量的内容，则可以使用 CSS 高度，但并非总是如此。如果您在手机上尝试上例（或调整浏览器窗口的大小），则会看到第二个框的内容将显示在框的外部。

如果这不是您想要的，请返回教程并学习另一个解决方案。
